Rotas publicas da API

1-Rota do proprietário da API

EX: http://www.racsstudios.com/api/v1

2-Rota para listar todos estabelecimentos cadastrados no sistema.

EX: http://www.racsstudios.com/api/v1/allapps

3-Rota para listar todos estabelecimentos com paginação. Quatro itens por pagina.

EX 1: http://www.racsstudios.com/api/v1/allapps
EX 2: http://www.racsstudios.com/api/v1/apps?page=2

4-Rota para selecionar um estabelecimento pelo "idApp do App..

OBS:Não trás os os comentários,


EX: http://www.racsstudios.com/api/v1/apps/id-desejado

5-Rota para buscar todos os comentários de um Estabelecimento. Rota com paginação.

EX: http://www.racsstudios.com/api/v1/comment/id-desejado

6-Rota para lista todos estabelecimentos de um segmento. Esta rota tambem possui paginação e ordenação

EX: http://www.racsstudios.com/api/v1/allapps/segmento?page=2&filter=elemento&order=ASC

7-Rota de pesquisa.

EX: http://www.racsstudios.com/api/v1/search?like=pesquisa EX: http://www.racsstudios.com/api/v1/search?like=PRa&filter=estrelas&order=ASC

Algumas rotas possuem filtro e ordenação teste eles-> filter=visualizacao //OPÇÕES: estrelas,custo e order=desc //OPÇOES: ASC
