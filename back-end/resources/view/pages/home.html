Quais cidades podem aderir ao sistema?

O sistema está disponível para cidades com até 200 mil habitantes. Mas fique tranquilo para cidades maiores o sistema logo ganhara uma versão para tratar por regiões do município. Entre em contato e enviar uma mensagem para saber mais sobre essa versão.

O que preciso fazer para ter o sistema em minha cidade?

Basta acessar nossa página de contato e solitar um reunião para avaliar as características de seu município.
